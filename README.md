# VidAI Studio — AI Video Generator

Professional AI Video Generator web application.

## Quick Start
1. Open `index.html` in any browser
2. Click "Sign Up Free" or "Log In"
3. Start generating AI videos!

## API Key
Gemini API Key is already configured.

## Files
- `index.html` — Main application
- `app.js` — API utilities
- `styles.css` — Extra styles
- `README.md` — Documentation

## Features
- Text to Video, Image to Video
- Multiple styles: Realistic, Anime, Pixar, 3D, Cyberpunk, etc.
- Dark/Light Mode
- User Authentication
- Dashboard
- Credits System
- Multi-language Support
